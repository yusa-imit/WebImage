## For Development

### Git clone & Dependancy install

For Development using npm
```bash
  git clone https://github.com/yusa-imit/WebImage.git
  cd WebImage
  npm install
  npm run start
```


using yarn

```bash
  git clone https://github.com/yusa-imit/WebImage.git
  cd WebImage
  yarn
  yarn start
```


### Compile


using npm
```bash
  npm run make
```


using yarn
```bash
  yarn make
```

### Publish


using npm
```bash
  npm run publish
```



using yarn
```bash
  yarn run publish
```
